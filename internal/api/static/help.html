SafeSky Help

Quick start

  1. Откройте Settings и добавьте подписку или один server URI.
  2. Выберите сервер в списке и нажмите подключение.
  3. Проверьте статус на главном экране или в меню трея.

VLESS, Trojan, Hysteria2, WireGuard

Это форматы доступа к серверу. Обычно провайдер выдаёт ссылку вида vless://..., trojan://..., hysteria2://... или конфигурацию WireGuard.

Profiles

Профили сохраняют набор routing rules и выбор сервера. Используйте их для разных сценариев: работа, браузер, стриминг.

Troubleshooting

  • Если сервер не подключается, запустите Diagnostics.
  • Если DNS/IP leak test показывает риск, переключите сервер или включите строгую защиту DNS.
  • Если hotkey занят другим приложением, выберите другое сочетание в Settings.

FAQ

Можно закрыть окно? Да, по умолчанию окно скрывается в трей, а клиент продолжает работать.

Как быстро отключиться? Используйте меню трея или Ctrl+Alt+P.
