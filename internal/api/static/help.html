SafeSky Help

Quick start

  1. Откройте Settings и добавьте подписку или один server URI.
  2. Выберите сервер в списке и нажмите подключение.
  3. Проверьте статус на главном экране или в меню трея.

Если это первый запуск, пройдите onboarding wizard: он предложит телеметрию, источник сервера и короткий connection test.

Protocols

Поддерживаются VLESS, Trojan, Shadowsocks, Hysteria2, TUIC, WireGuard и VMess. Обычно провайдер выдаёт ссылку вида vless://..., trojan://..., hy2://... или конфигурацию WireGuard.

Для VLESS доступны TCP, WebSocket, gRPC, HTTP/2 и HTTPUpgrade transports. Reality требует pbk, а XTLS Vision не совмещается с multiplexing.

Subscriptions

Подписки должны использовать HTTPS. SafeSky отклоняет HTTP redirects, шифрует metadata на диске и импортирует только поддерживаемые server URI.

Profiles

Профили сохраняют routing rules, DNS/settings state и выбор сервера. Используйте их для разных сценариев: работа, стриминг, игры, поездки.

Routing

Routing rules бывают domain, IP/CIDR, process и geosite. Action может быть proxy, direct или block. Remote DNS идёт через туннель, direct DNS используется для локального inbound.

Kill switch

Kill switch работает как fail-close: при падении туннеля блокировка может остаться активной до явного отключения или восстановления. Включайте его только если утечка хуже временной потери сети.

Troubleshooting

  • Если сервер не подключается, запустите Diagnostics и проверьте, свободны ли порты 10807, 9090 и API port.
  • Если DNS/IP leak test показывает риск, проверьте routing rules и DNS settings.
  • Если TUN не стартует, запустите SafeSky as administrator и дождитесь Wintun cleanup.
  • Если hotkey занят другим приложением, выберите другое сочетание в Settings.

FAQ

Можно закрыть окно? Да, по умолчанию окно скрывается в трей, а клиент продолжает работать.

Как быстро отключиться? Используйте меню трея или Ctrl+Alt+P.

Отправляется ли telemetry? Только после opt-in. Usage events и crash reports включаются отдельно.
